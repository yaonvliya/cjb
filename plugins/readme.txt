本文档将详细解释plugins目录下的所有插件名称和用途
	
>>  插件名称：cropper
	功能说明：图像裁剪插件(引入cropper.min.css、cropper.min.js)
	使用规范：依赖jquery，参考：http://fengyuanchen.github.io/cropper/


>>  插件名称：echarts
	功能说明：百度提供的图表插件-功能非常强大(引入echarts.min.js)
	使用规范：参考:http://echarts.baidu.com/index.html


>>  插件名称：viewer
	功能说明：相册图片图片展示特效(引入viewer.min.css、viewer.min.js)
	使用规范：依赖jquery


>>  插件名称：fullcalendar
	功能说明：日历日程插件-用日历的形式直观的展示了日程安排、代办事宜等事件(引入fullcalendar.css、fullcalendar.print.css、fullcalendar.min.js、moment.min.js)
	使用规范：依赖jquery,参考：http://www.fullcalendar.io/；中文说明文档:https://www.helloweba.com/view-blog-231.html


>>  插件名称：layui
	功能说明：经典模块化前端框架(引入layui.css、layui.js)
	使用规范：独立模块，不依赖任何库，可以单独作为框架使用，参考：http://www.layui.com/demo/


>>  插件名称：plyr
	功能说明：简单漂亮的HTML5视频音频播放插件(引入plyr.css、plyr.js)
	使用规范：参考：https://plyr.io/


>>  插件名称：sparkline
	功能说明：迷你型图表插件(引入jquery.sparkline.min.js)
	使用规范：依赖jquery,参考：https://omnipotent.net/jquery.sparkline/#s-about


>>  插件名称：xheditor
	功能说明：富文本编辑器(引入xheditor-1.2.2.min.js、zh-cn.js)
	使用规范：依赖jquery,参考：http://xheditor.com/manual.html


>>  插件名称：zTree
	功能说明：树形菜单(引入jquery.ztree.all.min.js、zTreeStyle.css或metroStyle.css)
	使用规范：依赖jquery,参考：http://www.treejs.cn/v3/main.php#_zTreeInfo
